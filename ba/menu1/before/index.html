Exercice 1

Analyser et corriger

Après avoir analyser les problèmes d'accessibilité propres à l'implémentation de ce menu, corriger cette implémentation.

A propos

Cet exercice est basé sur le menu de navigation Animated Menu (nouvelle fenêtre) de Una Kravets.

Avec les modifications du code original :

  • l'élément .menu-toggle (burger icon) a été déplacé dans l'élément nav ;
  • l'attribut role="navigation" a été déplacé de l'élément ul à l'élément nav.
