Écrire des tests d'accessibilité

Partie 1 : tester via CSS

Dans cette page :

  1. Créer une bordure verte de 5px les images avec un attribut alt ;
  2. Créer une bordure rouge de 5px les images sans attribut alt ;
  3. Créer une bordure jaune de 5px les liens de téléchargement de fichiers PDF.

Note : utilisez le fichier edit/a11y-tests.css lié à cette page pour réaliser ces exercices.

Partie 2 : tester via Javascript

Dans la console de cette page :

  1. Lister les tableaux de données non titrés ;
  2. Lister les champs de saisie et les attributs significatifs associés (aria-labelledby, aria-label, id et élément label associé, title) ;
  3. Lister les titres de la page afin de vérifier la hiérarchie des titres ;
  4. Lister les éléments dont la taille de police est exprimée en pixel.

Note : utilisez le fichier edit/a11y-tests.js lié à cette page pour réaliser ces exercices.
